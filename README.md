# Saqib Visuals | Digital Agency

A futurist, high-performance tech agency landing page built with **Next.js 15**, **Tailwind CSS v4**, and **Framer Motion**.

## 🚀 Overview

Saqib Visuals is a high-end engineering and design agency specializing in scalable architectures, AI integration, and modern web ecosystems. This landing page showcases a premium visual identity with a focus on interactive 3D aesthetics and smooth user experiences.

## ✨ Key Features

- **High-Performance Animations**: Optimized GPU-accelerated motion using Framer Motion.
- **Interactive System Architecture**: Dynamic SVG diagram visualizing the full-stack ecosystem.
- **Magnetic Components**: Tactile button feedback for a premium interactive feel.
- **Custom Cursor**: Reactive GL-style cursor for enhanced navigation.
- **Glassmorphism UI**: Modern frosted-glass 3D abstract shapes.
- **Responsive Design**: Fully optimized for mobile, tablet, and ultra-wide desktops.
- **Built-in SEO**: Comprehensive meta tags, OG imagery, and search engine optimization.

## 🛠 Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: motion/react (Framer Motion)
- **Icons**: Lucide React / React Icons
- **Fonts**: Inter, Outfit, JetBrains Mono

## 📦 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
